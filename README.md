# 🍽️ Diigiure Smart Menu — v3 (HTML Pure)

## هيكل المشروع

```
diigiure-v3/
│
├── index.html        ← واجهة العملاء (قائمة الطعام)
├── data.js           ← ✏️ هنا فقط تعدّل بيانات المتجر
│
└── admin/
    └── index.html    ← لوحة الإدارة (بدون تسجيل دخول)
```

## ✅ كيفية النشر

1. ارفع المجلد بالكامل على أي استضافة
2. افتح `index.html` — تشتغل القائمة فوراً
3. افتح `admin/index.html` — تشتغل لوحة الإدارة

**لا يحتاج**: Node.js، PHP، قاعدة بيانات، أي سيرفر

## ✏️ تخصيص المتجر

افتح `data.js` وعدّل:
- اسم المتجر والوصف
- رقم الواتساب
- الألوان (accentColor + heroGradient)
- الفئات والمنتجات

## 🔄 التزامن بين الإدارة والقائمة

- التعديلات في لوحة الإدارة تُحفظ في `localStorage`
- القائمة تقرأ من `localStorage` أولاً ثم من `data.js`
- لنشر تغييرات دائمة: اضغط "تصدير" في لوحة الإدارة (قريباً)

## 📱 النشر على GitHub Pages

1. ارفع المجلد على GitHub
2. اذهب إلى Settings → Pages
3. اختر الـ branch الرئيسي
4. الموقع يعمل على: `https://username.github.io/repo-name/`

## 📱 النشر على Netlify

- اسحب المجلد مباشرة على netlify.com/drop
- يعمل فوراً بدون إعداد

# 🍽️ Diigiure Smart Menu

## كيف يعمل؟

```
data.js  ←──────────────────────────────────────┐
   ↓                                             │
index.html   (يقرأ البيانات ويعرضها للزبائن)    │ save.php
                                                  │ (يحفظ التعديلات)
admin/index.html  (لوحة الإدارة تعدّل البيانات) ─┘
```

**أي تغيير في لوحة الإدارة → يُحفظ في data.js على السيرفر → يظهر لجميع الزوار فوراً**

---

## 📁 الملفات

```
diigiure-v4/
├── index.html          ← 👥 قائمة العملاء
├── data.js             ← 📦 قاعدة البيانات (ملف JS)
├── admin/
│   ├── index.html      ← ⚙️ لوحة الإدارة
│   └── save.php        ← 💾 حفظ data.js على السيرفر
└── README.md
```

---

## 🚀 النشر

### 1. استضافة PHP (cPanel, Hostinger, etc.)
ارفع كل الملفات على `public_html/` عبر File Manager أو FTP.  
**يعمل فوراً.**

### 2. GitHub Pages (بدون PHP)
ارفع الملفات كما هي — القائمة تعمل.  
لوحة الإدارة تعمل لكن زر "حفظ" سيُنزّل `data.js` الجديد بدلاً من الرفع التلقائي.  
ارفع الملف المُنزَّل يدوياً على GitHub لتظهر التغييرات.

### 3. Netlify
اسحب المجلد على netlify.com/drop — القائمة تعمل.  
لأتمتة الحفظ: استخدم Netlify Functions أو ارفع data.js يدوياً.

## 🔐 كلمة المرور

الافتراضية: `diigiure2024`

لتغييرها: افتح `admin/save.php` وغيّر:
```php
define('ADMIN_PASSWORD', 'كلمة_مرورك_الجديدة');
```
ثم غيّر نفس الكلمة في لوحة الإدارة (الإعدادات → كلمة مرور الإدارة).
